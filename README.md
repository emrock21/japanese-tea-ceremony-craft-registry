# Japanese Tea Ceremony Craft Registry

This registry documents traditional craftsmanship used in the Japanese tea ceremony.  
Each object in the tea room — chawan, chashaku, natsume, mizusashi, kama — is shaped by specific techniques, materials, and lineages that carry cultural and spiritual meaning.

## Purpose

The goal of this contract is to preserve essential tea ceremony craft techniques in a permanent and verifiable format.  
Many of these practices survive through apprenticeship and family workshops, and some are vulnerable as artisan communities shrink.

By recording selected techniques on-chain, the registry protects cultural knowledge from loss or distortion and keeps the connection between technique, material, and ritual purpose.

## Why this contract matters

The tea ceremony is not only a performance. It is a material culture built on centuries of craftsmanship.  
Bowls, utensils, and containers are created with intention, symbolism, and regional identity.

This registry helps by:

- preserving verified information in a neutral and permanent format  
- keeping cultural context attached to each technique  
- offering a reference for artisans, researchers, and cultural institutions  
- preventing the simplification or misrepresentation of traditional tea crafts  

## What makes this registry unique

The registry focuses on the craft process itself.  
It treats shaping, carving, firing, lacquering, and forging as cultural assets worth preserving.

The contract is intentionally minimal.  
It records only the essential information needed to identify and contextualize each technique.

## How it works

Each entry includes:

- region  
- workshop  
- technique name  
- material type  
- cultural note  
- ritual or traditional use  

A simple like and dislike system allows basic community feedback.
